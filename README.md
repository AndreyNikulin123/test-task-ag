# Aiti Guru — Test Task Frontend

Одностраничное приложение на **React 18 + TypeScript** c использованием публичного API **DummyJSON Auth / Products**.

## Запуск проекта

```bash
npm install
npm run dev
```

Приложение будет доступно в браузере (по умолчанию `http://localhost:5173`).

Рекомендуемые данные для входа:

- **Логин**: `emilys`
- **Пароль**: `emilyspass`

Для выполнения задания использовался Cursor AI (вёрстка макета из Figma с помощью MCP, логика работы приложения), чат на сайте Claude AI (Sonnet 4.6) для устранения недочётов верстки и логики.
